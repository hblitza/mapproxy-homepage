Alle Versionen von MapProxy werden auf der PyPI Projektseite zum Download bereitgestellt. Neben der aktuellen Version finden Sie hier auch ein Archiv mit allen offiziellen Versionen. Inoffizielle Versionen und Beta-Versionen können auf dieser Seite abgerufen werden.

Ein manueller Download von MapProxy ist aber oft nicht notwendig. MapProxy kann direkt mit den Python-Werkzeugen easy_install oder pip auf Ihrem System installiert werden. Mehr Informationen über die Installation finden Sie in der Dokumentation im Kapitel Installation.

Versionen

Der aktuelle Stand der Entwicklung ist in der Hauptversion 1.x abgebildet. Alle neuen Funktionen werden in die jeweils nächsten Hauptversion (engl. major release) veröffentlicht. Für kleinere Änderungen oder Fehlerbeseitigungen wird MapProxy zudem in kleineren Versionen, mit so genannten Nebenversionsnummer (engl. minor release), z.B. 1.2.1, zum Download bereitgestellt.

Änderungen zwischen einzelnen Versionen werden in der Textdatei CHANGES.txt dokumentiert.

1.10.x

Die aktuelle Version wurde unter der Apache Software License 2.0 veröffentlicht.

Um die aktuelle Version zu installieren führen Sie folgenden Befehl aus:

pip install MapProxy

Alte Versionen

Sie können vorherige MapProxy-Versionen weiterhin installieren. Bei der Installation müssen Sie hierfür die entsprechende Versionsnummer angeben. So kann zum Beispiel MapProxy in der Version 1.1.1 installiert werden:

pip install 'MapProxy==1.1.1'

Oder die letzte Version der Version 1.2.x zu, aber nicht die neuere Version 1.3:

pip install 'MapProxy<1.2.99'

1.1.x und neuer

Seit der Version 1.1.x wurden alle Versionen wurde unter Apache Software License 2.0veröffentlicht.

1.0.x

Die MapProxy Version 1.0.1 und alle vorherige Versionen wurden unter der GNU AGPL License 3.0 veröffentlicht.

Bitte beachten Sie die Informationen zum Upgrade, wenn Sie MapProxy von Version 0.9 aktualisieren

0.9.x

Achtung: Das Format der Konfiguration hat sich zwischen den Versionen 0.8.x und 0.9.x geändert. Achten Sie darauf, dass das neue Formateingehalten wird.

Nightly

Das MapProxy Projekt stellt keine "Nightly Builds" von MapProxy bereit. Dies ist nicht notwendig, da Sie MapProxy direkt aus den aktuellen Entwicklungsquellen installieren können:

pip install https://github.com/mapproxy/mapproxy/tarball/master

Die Quellen sind ebenfalls für die einzelnen Versionen verfügbar. Zum Beispiel:

pip install https://github.com/mapproxy/mapproxy/tarball/1.4.x
